Инструкция по запуску:
    1. Создать python virtual environment
    2. Установить зависимости (pip install -r r.txt)
    3. Активировать venv
    4. Запустить скрипт с параметром url статьи (python main.py url)

Алгоритм:
    Сначала находим главный заголовок на странице, потом по свойству itemprop, под котором на большинстве сайтов
    лежит основной текст страницы (если это статья), потом переводим это в текст, форматрируем строку и записываем
    в файл.

Возможное развитие программы:
    Чтобы сделать программу более универсальной, нужно по-сути написать еще одну программу, которая будет брать весь
    текст с сайта и удалять все ненужное шаг за шагом, удаляя ненужные теги, скрипты, и т.д.
    Так же можно добавить больше настроек для кастомизации выходных файлов.
